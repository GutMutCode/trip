# 도쿄 3박 4일 짐 체크리스트

기준 일정: 2026.05.11(월) ~ 2026.05.14(목), 도쿄/가마쿠라/에노시마

## 짐 싸는 기준

- Day 1은 새벽 출발 후 나리타, 아키하바라, 도쿄역, 오모테산도 체크인까지 이어지므로 첫날 바로 꺼낼 물건은 작은 가방에 분리한다.
- Day 2는 하라주쿠/아오야마, 도쿄타워, 아자부다이, 시부야 쇼핑 중심이라 보조배터리와 편한 신발이 중요하다.
- Day 3은 가마쿠라 해안 산책과 에노시마 일정이 길어 바람막이, 편한 양말, 우산 또는 우비를 챙긴다.
- Day 4는 체크아웃 후 오모테산도 쇼핑과 나리타 이동이 있으므로 캐리어 여유 공간을 남긴다.
- 5월 도쿄는 낮에는 가볍고 밤/해안은 선선할 수 있으니 반팔 또는 얇은 상의에 바람막이를 겹쳐 입는 쪽이 맞다.

## 전자기기와 전원

- [X] 보조배터리

- [X] 일본용 돼지코 겸용 충전기
- [ ] 이어폰
- [ ] 고데기

주의:

- 보조배터리는 위탁수하물에 넣지 말고 직접 휴대한다.
- 보조배터리는 단자 테이프, 보호캡, 파우치 중 하나로 단락 방지한다.
- 고데기는 유선 제품이고 입력 전압이 `100-240V`인지 확인한다.
- 배터리 내장형 또는 무선 고데기는 항공사/공항 규정상 제한될 수 있으므로 가져가지 않는 쪽이 안전하다.
- 일본은 100V 전원이라 220V 전용 고데기는 성능이 제대로 나오지 않을 수 있다.
